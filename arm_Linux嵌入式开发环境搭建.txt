Arm+Linux嵌入式开发开发环境搭建

1.编译环境：

  Windows安装VMware虚拟机，下载地址：https://www.vmware.com/cn/products/workstation-pro/workstation-pro-evaluation.html（需要注册密匙，网上可以找到资源）
  
  虚拟机中安装Ubuntu，Ubuntu镜像下载地址：https://ubuntu.com/download/desktop
  
  Ubuntu中安装arm-none-eabi-gcc交叉编译环境，在/usr/local目录下新建arm文件夹，工具安装在/usr/local/arm目录下
 （需要 sudo chmod 777 /usr/local/arm增加arm文件夹的权限）

  1.Ubuntu中指令安装：sudo apt-get install  gcc-arm-linux-gnueabihf
                      sudo apt-get install  g++-arm-linux-gnueabihf安装arm-linux-g++

    该方法2020.6.20在Ubuntu系统中尝试安装不成功，

  2.网上下载交叉编译工具，网址：https://developer.arm.com/tools-and-software/open-source-software/developer-tools/gnu-toolchain/gnu-rm（arm官方网站）

    注意：该网站上下载的是 arm-none-eabi-gcc 交叉编译工具，不是arm-linux-gcc交叉编译工具，
   
    注意：64位的Ubuntu系统需要安装64位的交叉编译工具，

    下载把文件解压到/usr/local/arm文件夹， 
    
    参考：https://blog.csdn.net/wu10188/article/details/86542418

  3.交叉编译工具的区别，参考网址：https://www.cnblogs.com/deng-tao/p/6432578.html
    
    ！！！有时间的时候要仔细了解Linux交叉编译的原理、不同交叉编译工具的区别
	
  4.解压缩文件到指定文件夹： tar -jxvf archive_name.tar.bz2 -C /tmp/extract_here/	
	   
  Windows中建立共享文档（方便编译后文件的烧录？）
